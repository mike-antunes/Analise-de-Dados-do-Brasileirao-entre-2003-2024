# Análise de dados dos times do Brasileirão entre os anos de 2003 até 2024

- ### Resumo do projeto
 
O objetivo foi encontrar padrões que definem quais métricas são mais importantes para tornar um time campeão ou rebaixado. Foi feito análise de dados, comparações do desempenho dos times, os top 5 times com melhores ataques e defesas, utilização do modelo de Machine Learning **Random Forest** para nos trazer qual critério é mais importante para determinar as posições dos times no campenato entre ter um ataque goleador ou uma defesa que sofre menos gols, entre outras análises.
